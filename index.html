Deploy Your Website Using Git + Amplify

A beginner-friendly guide to host websites with CI/CD.

📌 What is AWS Amplify?

AWS Amplify is a powerful hosting service that allows you to deploy frontend applications with automatic CI/CD, HTTPS, and global CDN support. It connects directly with Git repositories like GitHub.

⚙️ Steps to Deploy Using Git + Amplify

1. Push Code to GitHub

Initialize Git, add files, commit, and push to GitHub repository.

2. Open AWS Amplify

Go to AWS Console and open Amplify Hosting service.

3. Connect Repository

Select GitHub and authorize AWS to access your repository.

4. Configure Build Settings

Amplify auto-detects build settings for static websites.

5. Deploy

Click deploy and your site will be live in minutes.

🔥 Features of Amplify

⚡ Fast CDN

Powered by global CDN for lightning-fast performance.

🔄 CI/CD

Automatic deployment on every Git push.

🔒 Secure

Free SSL and HTTPS enabled by default.
